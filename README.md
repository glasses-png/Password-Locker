# Author

# Description
Password Locker is a terminal run python application that allows users to store details i.e. usernames and passwords of their various accounts.

# SetUp/Installation
* python3.6
* pyperclip
* pip

### Cloning
* In your terminal:
      $ git clone https://github.com/glasses-png/Password-Locker/
      $ cd Password-Locker
